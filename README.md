# 🛸 Rick and Morty Explorer

A fan-made web app built with **React**, **TypeScript**, and **Tailwind CSS** to browse characters from the Rick and Morty universe.

## ✨ Features

- 🧬 Browse characters fetched from the [Rick and Morty API](https://rickandmortyapi.com/)
- 🔍 Open a detail panel with each character's info
- ❤️ Favorite and unfavorite characters

## 🛠 Tech Stack

- **React** + **TypeScript**
- **Tailwind CSS**
- **Rick and Morty API**

## 🚀 Getting Started

```bash
npm install
npm run dev
```
